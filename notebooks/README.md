# Notebooks

분석 흐름을 재현하기 위한 Jupyter Notebook 모음입니다.

| Notebook | Purpose |
| --- | --- |
| `bapdo-pork-soup-list.ipynb` | 공공데이터에서 돼지국밥 음식점 후보군을 추출합니다. |
| `bapdo-review-crawling.ipynb` | 네이버 블로그 리뷰를 수집하고 수집 현황을 저장합니다. |
| `bapdo-review-langchain.ipynb` | Gemini API와 LangChain으로 리뷰 기반 특징 라벨링을 수행합니다. |
| `bapdo-review-summarizer.ipynb` | 리뷰 요약 및 라벨링 실험을 수행합니다. |
| `bapdo-image-crawling.ipynb` | 음식점 이미지 크롤링 과정을 담고 있습니다. |
| `bapdo-image-similarity.ipynb` | CLIP 임베딩으로 국밥 이미지 간 유사도를 계산합니다. |
| `bapdo-naver-smartstore.ipynb` | 네이버 스마트스토어 리뷰 수집 실험을 담고 있습니다. |

`GOOGLE_API_KEY`가 필요한 노트북은 코드에 키를 직접 쓰지 않고 환경변수를 사용합니다.
